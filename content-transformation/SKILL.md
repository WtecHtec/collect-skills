# Content Transformation Skill
# 公众号内容转化技能

## 用途
将知乎、天涯等平台的热门帖子，转化为公众号原创爆文的完整工作流。
适用赛道：养老金/民生、健康养生、历史故事、情感、法律维权、职场副业、三农。

---

## 文件结构

```
content-transformation/
├── SKILL.md                  # 本文件，使用说明
```

---

## 使用流程（五步走）

```
原始素材（知乎/天涯帖子）
        ↓
  Step 1：分析情绪点          → 使用 prompts/01_analyze.md
        ↓
  Step 2：选差异化角度        → 使用 prompts/02_angles.md
        ↓
  Step 3：生成爆款标题        → 使用 prompts/03_titles.md
        ↓
  Step 4：生成写作大纲        → 使用 prompts/04_outline.md
        ↓
  Step 5：生成完整正文        → 使用 prompts/05_write.md
```

每步的输出作为下一步的输入，形成完整链路。

---

## 核心原则

1. **不搬运内容，只借鉴角度** — 原帖是选题情报，文章必须重新创作
2. **情绪优先** — 找到为什么这个话题会火，比内容本身更重要
3. **标题决定一半** — 先确定标题，再写内容
4. **口语化表达** — 适合微信阅读场景，避免书面八股

## 各赛道注意事项

| 赛道 | 内容来源建议 | 特别注意 |
|------|-------------|---------|
| 养老金/民生 | 人社部、社保局官网 | 数据需准确，引用政策原文 |
| 健康养生 | 丁香医生、健康时报 | 避免"治疗""根治"等违禁词 |
| 历史故事 | 百科、知乎历史区 | 注明"据史料记载"等表述 |
| 情感 | 知乎情感区、天涯 | 避免涉及具体真实人物 |
| 法律维权 | 裁判文书网、法律博主 | 加"本文不构成法律建议"免责声明 |
| 职场副业 | 知乎职场区、脉脉 | 数据收益不夸大 |
| 三农 | 农业部官网、三农博主 | 贴近真实农村生活 |
