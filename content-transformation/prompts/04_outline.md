# Step 4 — 生成文章大纲
# prompts/04_outline.md

## 使用场景
标题确定后，用此提示词生成完整写作大纲。大纲是正文写作的骨架，决定文章结构质量。

---

## Prompt 模板

```
你是公众号爆文写手，赛道：「{赛道}」。

确定标题：{最终选定的标题}
写作角度：{选定角度}
核心痛点：{核心痛点}
目标读者：{目标读者画像}
目标字数：{800 / 1000 / 1200}字

请生成完整写作大纲，结构要求如下：

【第一部分：开头钩子（约100字）】
说明：用什么场景/数据/问题开头，让读者在前3行就无法停下来
要求：提供具体的开头句式或场景描述

【第二部分：正文（3-5个段落）】
每个段落提供：
- 小标题（加粗，8-12字）
- 本段核心内容（写什么，30字说明）
- 本段写作要点（怎么写，20字说明）
- 建议字数

【第三部分：结尾引导（约100字）】
说明：如何收尾，引导读者点击"在看"或留言
提供：具体结尾句式

【配套要素】
- 建议配图位置（在哪个段落插图，配什么内容的图）
- 互动引导语（放在文末的一句留言引导）
- 适合加入的数据/案例类型（不提供具体数据，说明应补充什么类型的信息）
```

## 公众号文章结构公式

根据不同类型，选择对应结构：

**实用信息型**（养老金/法律/操作指南）
```
开头：数据或现象引入（让读者觉得"这说的就是我"）
正文：问题是什么 → 原因分析 → 解决方案（步骤化）→ 注意事项
结尾：总结+行动引导
```

**情感共鸣型**（情感/三农/故事）
```
开头：场景描写或真实对话（让读者代入）
正文：故事展开 → 情感高潮 → 转折或反思 → 普遍意义
结尾：升华+引发共鸣的问题
```

**知识科普型**（健康养生/历史/法律知识）
```
开头：打破常见误解（"你可能不知道……"）
正文：是什么 → 为什么 → 怎么做 → 特殊情况处理
结尾：一句话总结+收藏提示
```

## 输出示例

```
标题：《养老金不足1000元？国家这3种补贴，很多老人没领到》

【开头钩子】
场景式开头：描写一位退休老人拿到养老金短信通知时的心情，
具体金额写出现实感，引出"很多人不知道还有补贴可以领"的钩子。
开头句式参考：「上个月，我妈退休金到账了——897块。」

【正文段落】

▌小标题1：养老金为什么这么低？（约200字）
内容：解释城乡居民养老保险vs职工养老保险的缴费差异
要点：用数字对比，让读者一眼看懂差距原因

▌小标题2：补贴一：高龄津贴，80岁以上必领（约200字）
内容：高龄津贴的申领条件、金额范围、办理方式
要点：操作步骤清晰，强调"很多人不知道"增加紧迫感
配图位置：此处插图（政务服务APP截图示意）

▌小标题3：补贴二：困难老人生活补贴（约200字）
内容：申领条件和材料清单
要点：列表形式，易读易收藏

▌小标题4：补贴三：失能老人护理补贴（约200字）
内容：针对生活不能自理的老人，申领流程
要点：强调"很多家庭不知道这个"，情感代入

【结尾引导】
总结三种补贴，结尾句式：
「父母年纪大了，这些钱本来就是属于他们的。转给家里有老人的朋友，帮他们查一查。」
互动引导语：「你家父母的养老金够用吗？评论区说说。」

【配套要素】
- 建议补充数据：各省高龄津贴金额对比表
- 建议补充案例：一个普通家庭成功申领的流程故事
```
