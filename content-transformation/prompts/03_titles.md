# Step 3 — 生成爆款标题
# prompts/03_titles.md

## 使用场景
选定写作角度后，用此提示词生成高点击率标题。标题是决定流量的最关键因素。

---

## Prompt 模板

```
你是公众号爆款标题专家，赛道：「{赛道}」。

选定角度：{选定角度名称}
写法方向：{写法说明}
核心痛点：{核心痛点}
目标读者：{目标读者画像}

请生成 5 个标题，严格遵守以下规则：

【硬性要求】
- 字数：15-25字
- 不使用感叹号超过1个
- 不使用"震惊""万万没想到"等低质标题词
- 不涉及违禁词（医疗承诺、政治敏感、夸大收益）
- 必须让目标读者在3秒内产生点击欲望

【标题公式参考，至少覆盖3种】
① 数字+痛点型：《养老金不足1000元？这5种补贴你一定要申领》
② 对比反差型：《同样退休，为什么他多领2000元/月》
③ 疑问悬念型：《农村老人养老金这么低，国家为什么不管？》
④ 操作指南型：《手把手教你查养老金，3分钟搞定》
⑤ 情感共鸣型：《父母养老金900块，这个家庭是怎么撑过来的》

【额外说明】
对每个标题，注明：
- 使用了哪种公式
- 适合配什么类型的封面图（一句话）
```

---

## 变量说明

| 变量 | 来源 |
|------|------|
| `{赛道}` | 同前 |
| `{选定角度名称}` | Step 2 选定的角度 |
| `{写法说明}` | Step 2 对应角度的写法说明 |
| `{核心痛点}` | Step 1 输出 |
| `{目标读者画像}` | Step 1 输出 |

---

## 标题禁用词清单

以下词语会触发平台限流或用户反感，禁止使用：

| 类型 | 禁用词示例 |
|------|-----------|
| 夸张类 | 震惊、炸裂、疯了、万万没想到、不敢相信 |
| 医疗承诺 | 根治、治好、特效、秘方、偏方 |
| 收益夸大 | 月入过万、躺赚、暴富、一夜暴富 |
| 政治敏感 | 涉及具体领导人、政治事件的表述 |
| 绝对化 | 必须、一定要、100%、永远 |

---

## 优质标题自检清单

生成标题后，逐条检查：

- [ ] 目标读者看到后会停下来吗？
- [ ] 标题是否暗示了文章有"干货"或"答案"？
- [ ] 字数是否在15-25字之间？
- [ ] 是否包含1个核心关键词（方便搜索）？
- [ ] 是否避开了所有禁用词？

---

## 输出示例

```
标题1：《养老金不足1000元？国家这3种补贴，很多老人没领到》
- 公式：数字+痛点型
- 封面图：老人手持社保卡，背景政务大厅

标题2：《同样交了15年社保，城里退4500，农村退900，差在哪》
- 公式：对比反差型
- 封面图：城市楼盘vs农村田野的拼图对比

标题3：《父母养老金每月只有800块，我们姐弟三个是怎么商量的》
- 公式：情感共鸣型
- 封面图：一家人围桌吃饭的温馨场景

标题4：《3分钟查清楚：父母到底能领多少养老金》
- 公式：操作指南型
- 封面图：手机截图展示查询界面

标题5：《2025年农村养老金会涨吗？看完这篇心里有数》
- 公式：疑问悬念型
- 封面图：日历翻页，标注2025年
```
