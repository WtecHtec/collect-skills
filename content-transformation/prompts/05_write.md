# Step 5 — 生成完整正文
# prompts/05_write.md

## 使用场景
大纲确定后，用此提示词生成可直接发布的完整正文。

---

## Prompt 模板

```
你是公众号爆文写手，赛道：「{赛道}」，风格口语化，适合中老年读者。

标题：{最终标题}
写作大纲：
{粘贴Step 4完整大纲}

请按大纲写出完整文章，严格遵守以下要求：

【语言风格】
- 口语化，像和老朋友聊天，不用书面语
- 句子短，每句不超过30字
- 多用"你""咱们""大家"拉近距离
- 多用真实场景和具体数字，不用模糊表达

【结构排版】
- 每个正文段落前加粗小标题（格式：▌小标题）
- 每段不超过200字，超过则拆分
- 关键信息用【】括号标注
- 数字和金额用阿拉伯数字

【内容要求】
- 数据类信息：用"据xx显示""参考xx规定"等表述，不要编造具体数字
- 操作步骤：用"第一步/第二步"清晰列出
- 情感内容：真实自然，不要矫情煽情
- 结尾必须有互动引导语（引导留言或转发）

【禁止事项】
- 不使用"震惊""不敢相信"等夸张词
- 不做医疗/法律/收益方面的绝对承诺
- 不抄袭原始素材的句子，全部重新表达
- 不超过{目标字数}字
- 文章中不能出现破折号、emoji图标
```

---

## 写完后的质检清单

发布前逐条检查，全部通过才能发：

**内容质量**
- [ ] 文章有没有提供真实有用的信息？
- [ ] 数据和政策表述有没有来源依据？
- [ ] 和原始素材的重合度是否低于20%？

**语言风格**
- [ ] 读起来像真人写的，还是AI味很重？
- [ ] 有没有长句（超过30字）需要拆分？
- [ ] 有没有书面用语需要换成大白话？

**违规风险**
- [ ] 有没有医疗承诺类表述？
- [ ] 有没有政治敏感内容？
- [ ] 有没有夸大收益或效果的描述？

**传播要素**
- [ ] 开头前3行能否让人继续读下去？
- [ ] 结尾有没有引导转发/留言的语句？
- [ ] 封面图是否准备好？

## 润色优化提示词（可选）

文章写完后，如果觉得还不够好，可以追加：

```
请帮我优化这篇文章，重点改进：
1. 开头改得更有吸引力，让人在第一句就想继续读
2. 把所有书面语换成口语表达
3. 结尾的转发引导语改得更自然，不要太硬

原文：
{粘贴文章}
```

## 不同赛道的语气模板

**养老金/民生赛道**
```
开头语气：实在、接地气
「上个月我妈养老金到账了，897块。她跟我说，够买菜了。我听完心里难受了很久。」
```

**健康养生赛道**
```
开头语气：轻松、有点小惊讶
「我有个习惯坚持了三十年，结果医生说，这个习惯正在悄悄伤害我的胃。」
```

**历史故事赛道**
```
开头语气：悬念感、引人好奇
「1945年那个下午，他做了一个决定，史书从未提过，但改变了很多人的命运。」
```

**情感赛道**
```
开头语气：真实、有代入感
「我妈问我，你们两个吵架了吗？我说没有。她说，那你怎么看起来不开心。」
```
