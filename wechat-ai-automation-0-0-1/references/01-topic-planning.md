# 01 - 选题策划阶段

本文档包含选题策划阶段（Step 1-5）的所有详细prompt模板。

---

## Step 1-2: 热点监测与筛选

### 使用场景
- 用户提供了具体热点/话题
- 用户只给了一个大致方向
- 用户完全没有想法

### Prompt模板

```
你是公众号选题策划专家。

我的公众号定位：
- 目标用户：职场打工人、职场领导（25-45岁）
- 内容方向：职场技能提升、管理经验、团队协作、职业发展
- 调性：实用、专业、有洞察、接地气

热点/话题方向：
[用户提供的热点或话题]

任务：评估这个热点是否适合做选题，或者生成相关的5-10个选题方向。

评估维度：
1. 用户匹配度（1-10分）：是否是目标用户关心的
2. 传播潜力（1-10分）：话题讨论度、情绪共鸣
3. 切入角度：我们可以从什么独特角度切入
4. 合规性评估（重要！）：
   - 是否会涉及不实信息（未证实的热点不能做）
   - 是否会煽动阶层对立（打工人vs老板、员工vs领导等需特别注意）
   - 是否会贩卖焦虑（如果是，必须能给出解决方法）
   - 是否涉及敏感话题（政治/宗教/血腥暴力等）
5. 风险评估（低/中/高）：综合合规风险
6. 时效性（强/中/弱）：热度能持续多久

输出格式：
热点：[热点名称]
匹配度：X分  传播潜力：X分
切入角度：[具体角度]
合规性：[详细说明是否合规，有无风险]
风险：低/中/高
时效性：强/中/弱
推荐理由：[说明]

如果不符合合规要求，必须明确指出并建议替代方向。
如果适合且合规，继续下一步生成具体选题。
```

### 输出示例

```
热点：年底绩效考核季讨论
匹配度：9分  传播潜力：9分
切入角度：从打工人和管理者双重视角看绩效考核，给出实用应对方法
合规性：符合规范（客观讨论，不煽动对立，提供方法）
风险：低（避免阶层对立，聚焦方法论）
时效性：强（年底季节性话题）
推荐理由：完美匹配目标用户（打工人关心被考核、领导关心怎么考核），痛点明确，有很强的实用价值

建议进入下一步生成具体选题方向。
```

## Step 3: 选题方向生成

### 使用场景
Step 2确认热点适合后，生成3个不同风格的选题供用户选择。

### Prompt模板

```
你是爆款选题策划专家。

热点：[Step 2确定的热点]
切入角度：[Step 2输出的角度]

任务：基于这个热点，生成3个不同风格的选题方向。

选题要求：
1. 标题：吸引但不标题党（20字内）
2. 核心痛点：击中职场打工人或职场领导的什么痛点
3. 内容角度：
   - 方向1：实用技能+方法论（干货工具型）
   - 方向2：管理经验+案例分享（领导力型）
   - 方向3：职场洞察+深度思考（认知提升型）
4. 预期传播：预估阅读量级别、转发潜力
5. 合规性：是否符合微信推荐规则

输出3个选题方向，每个包含：
- 标题（3个备选标题）
- 副标题/摘要
- 核心大纲（3-5个要点）
- 金句方向（3-5句）
- 配图建议
- 为什么这个角度能打动读者
```

### 输出示例

```
方向1：干货工具型
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

标题备选：
1. 年底绩效面谈，这5个技巧让你多拿30%奖金
2. 绩效考核季：打工人自救指南（附话术模板）
3. 我用这3个方法，连续5年拿A级绩效

副标题：
一份给职场打工人的绩效考核实战手册

核心大纲：
1. 绩效考核前的准备：如何梳理年度成果（附模板）
2. 绩效面谈的沟通技巧：3个话术让领导记住你的贡献
3. 如果绩效不理想：申诉流程和改进计划
4. 管理者视角：理解考核逻辑，找到突破口
5. 长期规划：如何建立个人工作档案

金句方向：
1. "绩效考核考的不是你做了什么，而是领导知道你做了什么。"
2. "准备工作做在平时，面谈只是临门一脚。"
3. "会说比会做更重要，会总结比会说更重要。"
4. "绩效不是和同事比，是和你自己的目标比。"
5. "A级绩效的秘密：让你的成果可量化、可证明。"

配图建议：
- 封面：职场人物在办公室，专业、自信的氛围
- 文中：绩效准备清单、面谈话术模板、工作成果整理表格

为什么这个角度能打动读者：
提供实用工具和具体方法，解决打工人年底最头疼的问题。有模板、有话术、有步骤，拿来就能用。

预期传播：5万+ 阅读，高收藏率（实用工具类）
合规性：完全符合，正向价值观，实用干货

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方向2：领导力型
[管理者视角，如何公平高效地做绩效考核]

方向3：认知提升型
[深度解析绩效考核背后的逻辑，提升职场认知]
```

### 用户选择后的操作

询问用户："这3个方向你更喜欢哪一个？或者需要我调整哪个方向？"

等用户确认后，进入Step 4。

## Step 4: 提取关键词

### 使用场景
用户选定选题方向后，提取SEO关键词。

### Prompt模板

```
选题：[用户选定的选题]

任务：提取关键词用于SEO优化和内容创作。

请输出：
1. 核心关键词（3-5个）：如"职场"、"职场打工人成长"、"30岁"
2. 长尾关键词（5-10个）：如"30岁职场职场打工人焦虑"、"独立职场打工人成长"
3. 情绪关键词（3-5个）：如"焦虑"、"清醒"、"成长"
4. 相关话题标签：如#职场成长 #独立职场打工人 #30岁

这些关键词需要在文章中自然出现3-5次。
```

### 输出示例

```
选题：30岁那年，我终于不再害怕35岁

核心关键词（3-5个）：
- 30岁
- 35岁危机
- 职场职场打工人
- 年龄焦虑
- 个人成长

长尾关键词（5-10个）：
- 30岁职场职场打工人成长
- 35岁职场危机应对
- 职场打工人年龄焦虑
- 职场职场打工人如何面对年龄压力
- 30-35岁职业规划
- 职场职场打工人自我提升
- 克服年龄焦虑的方法
- 职场中年职场打工人成长

情绪关键词（3-5个）：
- 焦虑
- 恐慌
- 觉醒
- 平静
- 从容

相关话题标签：
#职场职场打工人 #30岁 #年龄焦虑 #个人成长 #职场成长

使用建议：
- 在文章标题、摘要、开头、结尾自然出现核心关键词
- 长尾关键词分散在文章各段落
- 情绪关键词用于描述心路历程
- 话题标签添加到文章发布设置中
```

## Step 5: 文章结构设计

### 使用场景
关键词提取完成后，设计完整的文章框架。

### Prompt模板

```
选题：[选题名称]
关键词：[关键词清单]
风格：[用户选定的风格：暖心共鸣型/干货价值型/犀利清醒型]

任务：设计文章结构框架

要求：
1. 文章长度：1000字以上（根据选题复杂度灵活调整，可以是1500字、3000字或更长）
2. 结构：
   - 开头（100-300字）：真实故事/场景引入，制造共鸣
   - 主体（根据内容需要）：3-5个核心观点，每个观点包含：
     * 小标题（吸引人的）
     * 故事/案例（200-300字）
     * 扎心真相/洞察（100-200字）
     * 实用方法/干货（200-300字）
   - 结尾（100-300字）：情感升华 + 互动引导

输出详细大纲：

标题：[最终标题 + 2个备选]

一、开头（500字）
场景/故事：[具体场景描述，要有画面感]
钩子：[吸引读者继续读下去的点]

二、主体部分

观点1：[小标题]
- 故事：[故事梗概，要真实可信]
- 真相：[核心洞察，要扎心]
- 方法：[3个可执行的具体建议]

观点2：[小标题]
[重复结构]

观点3：[小标题]
[重复结构]

三、结尾（500字）
- 情感升华方向：[如何升华主题]
- 互动话题：[什么问题能引发评论讨论]

四、文章金句（10句）
[列出10个可以用在文章中的金句]

五、配图建议（5-8张）
[每个部分需要什么样的配图]
```

### 输出示例

```
标题：年底绩效面谈，这5个技巧让你多拿30%奖金
备选1：绩效考核季：打工人自救指南（附话术模板）
备选2：我用这3个方法，连续5年拿A级绩效

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

一、开头（100-300字）

场景/故事：
去年12月，我坐在会议室里，看着领导在绩效评估表上写下"B"。那一刻，我的心凉了半截。

明明这一年我加了无数次班，做了3个重点项目，为什么还是B？

走出会议室，我下定决心：今年，一定要拿A。

钩子：
一年后，我坐在同样的会议室里，领导在评估表上写下"A"，还说："你今年的表现很突出。"

这一年我做了什么？其实就是掌握了5个技巧。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

二、主体部分

观点1：绩效考核前的准备——你的成果，领导不一定知道

故事：
很多人以为"我做得好，领导自然看得到"。但现实是，领导一年要管10个人，每个人做了什么，他真的记不清。

去年的我就是这样，埋头苦干，从不主动汇报。结果到了年底，领导只记得我做过1个项目，其他2个完全忘了。

真相：
绩效考核考的不是你做了什么，而是领导知道你做了什么。

方法：
1. 建立工作日志：每完成一个重要任务，立刻记录（时间/成果/数据）
2. 月度主动汇报：每月给领导发一封邮件，总结本月成果
3. 准备绩效材料：提前2周整理全年成果清单，用数据说话

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

观点2：绩效面谈的沟通技巧——3个话术模板

[详细方法]

观点3：如果绩效不理想——申诉和改进

[详细方法]

观点4：理解领导的考核逻辑

[深度分析]

观点5：长期规划——建立个人工作档案

[长期建议]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

三、结尾（100-300字）

情感升华方向：
绩效考核，不只是年底那一次面谈，而是全年365天的积累。

从B到A，我学会的最重要的事是：主动让自己被看见。

不是为了讨好领导，而是为了让自己的付出有价值，让自己的努力被认可。

希望今年年底，你也能拿到满意的绩效。

互动话题：
你在绩效考核中遇到过什么问题？今年的绩效目标是什么？欢迎在评论区分享，我们一起交流。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

四、文章金句（10句）

1. 绩效考核考的不是你做了什么，而是领导知道你做了什么
2. 准备工作做在平时，面谈只是临门一脚
3. 会说比会做更重要，会总结比会说更重要
4. 绩效不是和同事比，是和你自己的目标比
5. A级绩效的秘密：让你的成果可量化、可证明
6. 不要等到年底才想起要表现，那已经晚了
7. 主动汇报不是邀功，是对自己工作的负责
8. 数据会说话，但你要先整理出来
9. 绩效面谈，是展示自己的舞台，不是等待审判的法庭
10. 从B到A，距离没有你想的那么远

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

五、配图建议（5-8张）

1. 封面图：职场人物在办公桌前，专业自信的氛围
2. 工作日志模板：表格形式，展示如何记录
3. 月度汇报邮件示例：截图或设计图
4. 绩效材料清单：信息图表
5. 面谈话术卡片：重点话术的卡片设计
6. 申诉流程图：流程示意图
7. 个人工作档案模板：表格或文档预览
```

### 输出后的操作

询问用户："这个结构你满意吗？需要调整哪里？"

确认后告诉用户："结构确认！接下来我会用AI生成完整初稿，大约需要15-20分钟。生成后，最关键的Step 7需要你花60-90分钟进行人工优化。"

然后进入Step 6。
