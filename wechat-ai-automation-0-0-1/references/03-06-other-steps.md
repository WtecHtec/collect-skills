# 04 - 发布优化阶段

## Step 10: 标题优化

### Prompt模板

```
文章主题：[主题]
目标用户：职场打工人、职场领导（25-45岁）

任务：生成10个不同风格的标题

标题类型（每种2个）：
1. 数字型：包含数字，如"3年时间，我从月薪5K到年薪30W"
2. 疑问型：引发好奇，如"30岁还没成功，是我不够努力吗？"
3. 对比型：强烈对比，如"同龄人年薪50W，我月薪5K，差距在哪？"
4. 共鸣型：情感共鸣，如"30岁的你，是不是也在假装坚强"
5. 冲突型：观点冲突，如"那些职场鸡汤，差点毁了我的30岁"

每个标题：
- 长度：15-25字
- 吸引但不标题党
- 包含关键词：[关键词]
- 符合账号调性
- 符合微信规范（不浮夸、不误导、与内容一致）

每个标题后面给出：
- 预估点击率（1-10分）
- 推荐理由（为什么这个标题能吸引点击）
- 合规性检查（是否符合微信标题规范）

最后推荐得分最高且合规的前3个。
```

## Step 11: 发布前检查清单

完整的合规检查清单（基于微信公众号推荐运营规范）：

```
发布前必查项（逐一确认）：

【第一关：基础合规】
[ ] 内容是原创的（不是转载、不是抄袭、不是洗稿）
[ ] 不是分组群发
[ ] 已选择"允许内容被推荐"

【第二关：真实性】
[ ] 所有事实都有依据，没有捏造
[ ] 没有偷换概念、断章取义、歪曲事实
[ ] 热点相关内容都已证实，不传播未证实信息
[ ] AIGC生成内容已深度人工加工（原创度50%以上）
[ ] 有明确的个人观点和独特洞察

【第三关：价值观（重要！）】
[ ] 没有煽动阶层对立（职场内容必查！）
   - 不写"打工人vs老板""员工vs领导"对立内容
   - 不煽动职场矛盾和阶层对立
   - 客观理性，提供解决方案
[ ] 没有贩卖焦虑（指出问题必须给出方法）
[ ] 传递正向价值观
[ ] 没有宣扬拜金主义、攀比、浪费
[ ] 没有网络暴力、侮辱嘲讽他人

【第四关：内容健康度】
[ ] 无低俗擦边内容（无性暗示、敏感部位描述）
[ ] 无暴力血腥画面
[ ] 无惊悚恐怖内容
[ ] 无伪科学、封建迷信

【第五关：标题规范】
[ ] 标题与内容一致，不误导
[ ] 标题没有浮夸描述（不用"震惊""绝对"等）
[ ] 关键信息完整，不故意制造悬念
[ ] 不是标题党

【第六关：用户体验】
[ ] 内容完整，不导流到微信外平台
[ ] 逻辑清晰，不是简单拼凑堆砌
[ ] 排版清晰美观
[ ] 配图清晰、无水印
[ ] 手机端预览效果良好
[ ] 不强制或诱导用户关注

【第七关：技术设置】
[ ] 原创声明已勾选
[ ] 话题标签已添加（如#职场成长）
[ ] 文章摘要已填写（80字左右）
[ ] 封面图已设置
[ ] 推送时间已设定（建议晚上21:00-22:00）

【第八关：营销限制】
[ ] 不是纯广告或营销推广
[ ] 不包含折扣优惠信息
[ ] 不诱导关注、不强制关注

【第九关：互动设置】
[ ] 文末互动话题已添加
[ ] 评论区已开启
[ ] 赞赏已开启（如有资格）

如果有任何一项回答"否"，必须修改后再发布。

特别提醒：
- 职场内容必须特别注意"阶层对立"问题
- 贩卖焦虑的内容必须给出具体解决方法
- AIGC内容必须深度人工加工，确保原创度
```

详细规范查看：`references/08-wechat-rules.md`

# 06 - 补充功能

## 评论管理

```
这是文章的评论：
[评论列表]

请帮我：
1. 分类：好评/中评/差评/提问
2. 为每条生成回复话术（30-50字，真诚温暖）
3. 找出3-5条优质评论建议置顶
4. 提取用户关心的问题作为选题方向

回复要求：真诚、温暖、专业

注意：
- 如果有负面评论涉及合规问题，需要特别标注
- 不要在回复中引发性别对立或其他敏感话题
```

## SEO优化

```
文章主题：[主题]
核心关键词：[关键词]

请优化SEO：
1. 写80字文章摘要（含关键词，吸引点击，符合规范）
2. 建议5个话题标签
3. 关键词布局建议（不影响阅读）
```

## 多平台分发

```
这是公众号文章（1000字以上）。

请改编成：
1. 小红书版本（1000字 + 10个金句）
2. 知乎版本（更客观理性）
3. 微博版本（300字精华 + 引流钩子）

保持核心观点一致，适配平台特点。

注意：
- 所有平台都要符合各自的社区规范
- 不同平台对性别话题的尺度可能不同
- 确保改编后的内容依然合规
```
