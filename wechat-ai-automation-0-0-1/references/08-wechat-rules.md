# 微信公众号推荐运营规范（完整版）

本文档整理了微信公众号推荐机制的所有规范要求，供创作时参考。

## 核心原则

**允许被推荐的前提**：
1. 创作者需主动选择"允许内容被推荐"
2. 内容必须符合推荐运营规范和公众平台运营规范
3. 转载和分组群发的内容将不会被推荐
4. 一旦取消推荐，无法重新选择允许推荐

**推荐生效场景**：
- 订阅号消息列表-看一看
- 文章页底部推荐
- 发现页-看一看入口推荐

---

## 一、平台鼓励的内容（优先推荐）

### 1. 有价值、有深度的原创内容
- 以传递知识、分享经验和展示个人观点为主要目的
- 有价值、有深度
- 必须原创

### 2. 优质阅读体验
- 精美的设计和排版
- 清晰的结构
- 舒适的视觉效果

### 3. 启发性、教育性或娱乐性内容
- 让读者在轻松阅读中收获知识或思考
- 有启发性
- 有教育价值
- 正向娱乐（不低俗）

### 4. 专业权威内容
- 金融、教育、医疗卫生、司法等专业人士发布
- 权威真实信息
- 专业知识

---

## 二、平台不希望看到的内容（不会被推荐）

### 1. 违法违规内容
- 违反法律法规
- 违反社会主义核心价值观

### 2. 不实内容
包括但不限于：
- 捏造或扭曲事实以吸引眼球博取流量
- 偷换概念、断章取义，歪曲事件事实
- 编造或传播未证实的信息
- 利用新技术（如AIGC）制作的虚假信息
- 未经考证的、无依据揣测的历史内容

### 3. 违背公序良俗内容
- 不尊重他人
- 令人极度不适
- 其他不良内容

### 4. 营销广告内容
- 仅以营销、广告为目的
- 缺乏实际价值的宣传和营销推广
- 虚假宣传

### 5. 恶意绕过平台检测
- 通过各种手段绕过平台检测、审核

## 三、详细运营规范（违反将不被推荐）

### 1. 恶意使用

**1.1 危害平台安全、扰乱秩序**
- 钻平台漏洞以赚取流量
- 假冒、仿冒机构、组织、媒体名称
- 破坏平台秩序的行为

**1.2 恶意违规及对抗**
- 使用技术手段规避检测
- 批量运营公众号
- 添加无关标签误导用户

### 2. 侵权内容

禁止：
- 发布、传播推广盗版内容
- 抄袭、搬运内容
- 混淆大IP/品牌，误导用户

### 3. 不实信息

禁止：
- 偷换概念、断章取义
- 编造或传播未证实的信息
- 利用AIGC制作的虚假信息
- 未经考证的历史内容

**特别注意**：
- 关于社会热点或公众关心的事件，必须基于事实
- 不得编造或传播未证实的信息
- 不得歪曲事实

### 4. 违背公序良俗

**4.1 恶搞灾难**
- 禁止恶搞重大自然灾害、意外事故、恐怖事件、战争等

**4.2 传播不良价值观**
禁止：
- 宣扬流量至上，展示违背伦理道德的生活
- 宣扬浪费、攀比、误导消费、拜金主义
- 实施或教唆"网络暴力"
- 鼓励PUA撩妹、撩汉等
- **煽动性别对立、性别歧视**（重要！）

**4.3 低级趣味、低俗擦边**
禁止：
- 露出/聚焦敏感部位的图片
- 突出身材特征和敏感部位
- 含有性暗示的描述
- AV女优/男优相关内容
- 描述人类性行为模式

**4.4 暴力、恶意谩骂**
- 宣扬暴力
- 恶意谩骂、侮辱嘲讽他人

### 5. 不良信息

**5.1 不规范医疗科普**
禁止：
- 未经证实的养生、食膳、锻炼偏方秘方
- 片面/夸大描述药品、医疗功效
- 不符中医理论的内容

**5.2 宣扬伪科学**
禁止：
- 无科学依据的食物相克
- 违背自然规律的内容
- 宣扬封建迷信

**5.3 引起不适的内容**

**5.3.1 血腥暴力**
- 死亡事故现场图
- 人体受到物理伤害的展示图
- 人体器官或脏器图（即使是医学科普）
- 动物撕咬进食特写

**5.3.2 惊悚恐怖**
- 恐怖灵异画面
- 真实涉及恐怖因素刑侦案件描述
- 都市灵异事件、民间悬疑故事

**5.3.3 误导类标题**
禁止：
- 以浮夸的描述，煽动人群做/不做某行为
- 标题无中生有部分信息
- 标题与文章内容完全不一致
- 标题与正文内容/中心思想无关

**5.3.4 关键信息缺失的标题**
禁止：
- 标题信息不完整，影响用户阅读
- 标题故意隐藏关键信息，制造悬念

**5.3.5 降低用户体验**
禁止：
- 简单拼凑的无意义无逻辑内容
- 视频内容异常（画质模糊、内容不完整）
- 制作混剪、拼接、P图误导用户
- 强制或诱导用户关注公众号

**5.4 导流内容**
禁止：
- 发布不完整内容进行导流
- 文章主要目的是导流至其他地方
- 内容信息不完整，影响用户体验

### 6. 营销推广

禁止：
- 仅为组织宣传、商业推广广告的文章
- 以促进销售为主要目的的广告内容
- 包含折扣优惠的内容
- 强制或提供奖励诱导用户关注

## 四、创作时的合规检查清单

### 基础合规
- [ ] 内容是原创的（不是转载、不是抄袭）
- [ ] 不是分组群发
- [ ] 已选择"允许内容被推荐"

### 内容价值
- [ ] 有传递知识、分享经验或展示个人观点
- [ ] 有实际价值和深度
- [ ] 排版清晰美观

### 真实性
- [ ] 所有事实都有依据，没有捏造
- [ ] 没有偷换概念、断章取义
- [ ] 热点相关内容都已证实，不传播未证实信息
- [ ] 如使用AIGC，已人工深度加工，保证原创度50%以上

### 价值观
- [ ] 传递正向价值观
- [ ] 没有煽动性别对立（重要！职场职场打工人内容需特别注意）
- [ ] 没有宣扬拜金主义、攀比、浪费
- [ ] 没有贩卖焦虑（指出问题后给出方法）
- [ ] 没有网络暴力、侮辱嘲讽他人

### 内容健康度
- [ ] 无低俗擦边内容
- [ ] 无性暗示、敏感部位描述
- [ ] 无血腥暴力画面
- [ ] 无惊悚恐怖内容
- [ ] 无伪科学、封建迷信

### 标题规范
- [ ] 标题与内容一致，不误导
- [ ] 不使用浮夸描述
- [ ] 关键信息完整，不故意制造悬念
- [ ] 不标题党

### 用户体验
- [ ] 内容完整，不导流到其他平台
- [ ] 逻辑清晰，不是拼凑堆砌
- [ ] 不强制或诱导用户关注

### 营销限制
- [ ] 不是纯广告或营销推广
- [ ] 不包含折扣优惠信息
- [ ] 不诱导关注

## 五、针对职场内容的特别提醒

### 必须避免的雷区

**1. 阶层对立类内容（绝对禁止）**
不能写：
- "打工人vs老板""员工vs领导"的对立话题
- "职场不公平"并煽动对立情绪
- "打工人vs资本家"的阶级矛盾内容

可以写：
- "职场打工人如何提升技能"（聚焦解决方案）
- "我从基层到管理的经历"（个人经历）
- "职场中的利益差异客观存在，我们可以..."（客观理性）

**2. 贩卖焦虑（需要给出方法）**
不能只写：
- "35岁职场危机怎么办"然后就结束
- "裁员潮来了"但不给方法

必须：
- 指出问题后，给出3个以上可执行的具体方法
- 传递希望和可能性
- 最终价值观是正向的

**3. 拜金主义、攀比**
不能写：
- "年薪多少才算成功"
- "35岁没有XX就是失败"
- 过度强调收入、职位、物质

可以写：
- "我如何从基层到年薪50W"（分享方法）
- "理性看待收入与价值"（价值观正向）

**4. 标题党**
不能写：
- "震惊！90%的职场人都..."（浮夸）
- "你绝对想不到的..."（故意隐藏信息）
- "不看后悔！"（强制性引导）

应该写：
- 具体、真实、与内容一致的标题
- 可以有吸引力，但不夸张、不误导

## 六、AIGC内容的特别要求

### 平台对AIGC的态度
- 不反对AI辅助创作
- 反对"低价值AIGC内容"（直接生成无人工加工）
- 要求内容有"信息增量"和"原创价值"

### 如何避免被判定为低价值AIGC

**1. 人工深度加工**
- Step 7的人工优化环节必不可少
- 加入真实个人经历和细节
- 确保原创度50%以上

**2. 信息增量**
- 不是简单汇总网上已有内容
- 必须有独特观点或新角度
- 必须有个人经验或方法论

**3. 个人观点**
- 明确的个人立场和思考
- 独特的洞察和分析
- 不是泛泛而谈

**4. 真实性**
- 不能用AI编造虚假故事
- 真实数据、真实案例
- 如使用虚构故事，需要合理且不误导

## 七、发布前终审清单（严格执行）

在点击"发布"前，必须逐项检查：

### 第一关：基础合规
1. 内容是原创的吗？
2. 已选择"允许被推荐"吗？
3. 不是转载或分组群发吗？

### 第二关：真实性
4. 所有事实都有依据吗？
5. 没有捏造、夸大、歪曲事实吗？
6. 热点相关内容都已证实吗？
7. AIGC生成内容已深度人工加工吗？

### 第三关：价值观
8. 没有煽动阶层对立吗？（职场内容必查）
9. 没有贩卖焦虑吗？（有问题有方法）
10. 传递的是正向价值观吗？
11. 没有宣扬拜金、攀比、浪费吗？

### 第四关：内容健康度
12. 无低俗擦边内容吗？
13. 无暴力血腥画面吗？
14. 无惊悚恐怖内容吗？
15. 无伪科学、迷信内容吗？

### 第五关：标题规范
16. 标题与内容一致吗？
17. 标题没有浮夸、误导吗？
18. 关键信息完整吗？

### 第六关：用户体验
19. 内容完整，不导流吗？
20. 排版清晰美观吗？
21. 不强制诱导关注吗？

### 第七关：营销限制
22. 不是纯广告或营销吗？
23. 不包含折扣优惠信息吗？

**如果有任何一项回答"否"，必须修改后再发布。**

## 八、违规后果

1. 内容不被推荐
2. 正在推荐的内容被取消推荐
3. 影响账号权重
4. 严重违规可能被限制功能

---

## 九、数据查看

发布后可在"内容分析-单篇群发"页面查看：
- 文章是否获得推荐
- 推荐带来的阅读量
- 推荐场景数据

通过数据优化创作策略。

---

## 十、总结：合规创作的核心要点

1. **必须原创**：不转载、不抄袭、不洗稿
2. **真实准确**：不捏造、不夸大、不误导
3. **价值观正**：正向、不对立、不焦虑
4. **内容健康**：不低俗、不暴力、不恐怖
5. **标题规范**：真实、完整、不误导
6. **用户体验**：完整、清晰、不导流
7. **非营销**：有价值、有深度、不是广告

**记住**：符合规范是推荐的前提，内容质量决定推荐效果。
