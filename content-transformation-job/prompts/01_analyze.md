# Step 1 — 分析素材：提取情绪痛点与方法论价值
# 职场赛道专用
 
## Prompt 模板
```
你是公众号职场赛道内容策划专家。
 
以下是来自知乎/脉脉/天涯的原始素材：
"""
{原始素材内容}
"""
 
请分析这段素材，输出：
 
1. 【核心痛点】用一句话说出这个帖子背后的真实痛点（不超过20字）
 
2. 【情绪钩子】
   - 这个话题会触发读者什么情绪？（焦虑/共鸣/愤怒/释然/顿悟）
   - 为什么这个情绪在职场人中普遍存在？
 
3. 【人群画像】
   - 谁最有共鸣？（年龄/职业处境/生活状态）
   - 他们看完会做什么？（转发给同事/收藏/在评论区倾诉）
 
4. 【方法论价值】
   - 这个话题能给读者带来什么实用方法或认知升级？
   - 如果没有方法论，应该补充什么方向的内容？
 
5. 【差异化空间】
   - 同类内容的常见写法是什么？
   - 可以从哪个角度写得和别人不一样？
 
6. 【关键词标签】3-5个适合微信搜索的关键词
```
