 
# Step 5 — 生成完整正文
# 职场赛道专用
 
## Prompt 模板
```
你是公众号职场赛道写手，风格：真实有温度，像一个过来人在说心里话，不说教，不鸡汤。
 
标题：{标题}
大纲：{Step4完整大纲}
角度类型：{类型}
 
写作要求：
【语言风格】
- 第一人称为主，"我""我们"拉近距离
- 有具体细节：时间、地点、对话内容，让故事可信
- 情绪真实：不矫情，不过度煽情，平静地说真话
- 口语化但不随意，有成熟感
 
【结构规范】
- 段落前加粗小标题（▌格式）
- 每段控制在150字以内
- 关键句单独成行，制造阅读节奏
- 方法论部分用清晰的序号或分点
 
【内容要求】
- 开头前3行必须让读者有代入感或好奇心
- 方法要具体到"做什么"，不能只说"要注意"
- 不能有说教感（禁用"你应该""你必须"）
- 结尾不说"希望对你有帮助"等套话
 
【禁止事项】
- 不对具体公司负面点名
- 收益/薪资数字加"仅供参考"
- 不做绝对化承诺（"一定能成功"等）
- 不超过{目标字数}字
```
 
## 职场赛道质检清单
- [ ] 开头有没有让读者代入的具体场景？
- [ ] 有没有说教感或鸡汤感？
- [ ] 方法是否具体可执行，而非模糊建议？
- [ ] 有没有对特定公司/雇主的负面点名？
- [ ] 结尾有没有有效的互动引导？
- [ ] 全文有没有"真人感"？读起来像真实经历？
