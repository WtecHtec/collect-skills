# Step 2 — 生成差异化角度
# 职场赛道专用
 
## Prompt 模板
```
你是公众号职场赛道内容策划专家。
 
分析结果：
- 核心痛点：{核心痛点}
- 情绪钩子：{情绪钩子}
- 人群画像：{人群画像}
- 方法论价值：{方法论价值}
- 差异化空间：{差异化空间}
 
生成4个差异化角度：
 
角度A【情感共鸣+方法论型】
切入：从一个真实场景/对话开头，引发强共鸣后给出方法
适合：话题情绪浓烈，读者需要被理解后被引导
 
角度B【反常识颠覆型】
切入：打破一个职场普遍误解，给出反直觉的真相
适合：话题有认知盲区，读者有"原来如此"的顿悟感
 
角度C【清单操作型】
切入：直接给N个可落地的具体方法，干货密度高
适合：话题实用性强，读者有明确的行动需求
 
角度D【真实案例拆解型】
切入：用一个完整的真实经历（改编）串联所有观点
适合：话题复杂，需要故事来承载方法论
 
每个角度提供：
- 切入句式示例（第一句话怎么写）
- 标题示例
- 预期读者行为
```
