
# Step 4 — 生成文章大纲
# 职场赛道专用
 
## Prompt 模板
```
你是公众号职场赛道写手。
 
标题：{标题}
角度类型：{共鸣方法/反常识/清单/案例}
核心痛点：{核心痛点}
目标读者：{目标读者}
目标字数：{900/1100/1300}
 
职场赛道大纲结构（按角度类型选择）：
 
【情感共鸣+方法论型】
开头：一个具体的职场场景或对话（让读者秒代入）
▌你是不是也有过这种感受？（共鸣铺垫）
▌为什么会这样？（原因分析，有认知深度）
▌我后来是怎么做的？（方法论，3-4个具体点）
▌现在回头看，最重要的是这件事（升华）
结尾：一句走心的话 + 留言引导
 
【反常识颠覆型】
开头：一个反直觉的结论直接抛出（制造好奇）
▌大多数人的误解是什么？（建立对立面）
▌真相其实是这样的（核心观点展开）
▌为什么会有这个误解？（分析根源）
▌知道真相后，你可以这样做（实操建议）
结尾：一个问题引发讨论
 
【清单操作型】
开头：一句话描述痛点场景（精准命中）
▌方法一：[具体可操作的建议]
▌方法二：...
▌方法三：...（共3-5个，每个有案例支撑）
▌最重要的一点（压轴干货）
结尾：收藏提示 + 留言引导
 
每段注明：核心内容（30字）+ 情绪/写作要点（20字）
配套：互动引导语 + 建议封面图风格
```
