
# Step 3 — 生成爆款标题
# 职场赛道专用
 
## Prompt 模板
```
你是公众号职场赛道标题专家。
 
选定角度：{角度}
核心痛点：{核心痛点}
目标读者：{目标读者}
 
生成5个标题，符合职场赛道风格：
- 有情绪张力（焦虑/共鸣/顿悟感）
- 有具体人群或场景
- 15-24字
- 禁用：成功学口号式表达
 
职场赛道常用标题公式（覆盖3种以上）：
① 危机共鸣型：《XX岁被裁，我是怎么熬过来的》
② 反常识型：《越努力的人，为什么越容易被裁？》
③ 清单干货型：《职场上最聪明的5件事，没人教你》
④ 真实经历型：《打工10年，我后悔没早知道这些》
⑤ 对比启示型：《同样入职3年，为什么他升了你没有》
 
每个标题注明：使用公式 + 封面图风格建议
```
