# Content Transformation — 职场赛道
# 适用：职场生存技巧、裁员/跳槽、AI+职场、副业变现、职场人际

## 赛道特征
- 受众：25-45岁城市职场人，尤其35岁危机人群
- 内容生命周期：中长（话题常青，但热点时效短）
- 核心价值：情感共鸣 + 可执行方法论
- 变现路径：流量主 + 知识付费课程 + 咨询

## 文件结构
```

│   ├── 04_outline.md     # 生成大纲：故事代入+方法干货

```

## 转化流程
```
原始素材（知乎/脉脉/天涯职场帖子）
        ↓
Step 1：提取情绪钩子 + 方法论价值
        ↓
Step 2：选角度（共鸣故事/方法清单/真实案例/警示）
        ↓
Step 3：生成标题（危机感+数字+反常识）
        ↓
Step 4：生成大纲（故事开头→问题分析→方法论→结尾升华）
        ↓
Step 5：生成正文（像真人经历，不像教科书）
```

## 核心原则
1. **真实感第一**：读者能感受到"这是真人写的"，有具体细节、真实情绪
2. **先共鸣后方法**：不要上来就给建议，先让读者觉得"说的就是我"
3. **方法要可落地**：每个建议都要具体到"第一步做什么"，不能只说"要努力"
4. **避免说教**：不要用"你应该""你必须"的口吻，用"我发现""后来我才明白"

## 素材来源优先级
1. 知乎职场话题高赞问答（情感最真实）
2. 脉脉职场吐槽（最新职场现状）
3. 天涯职场版长帖（故事性强，适合改写）
4. Boss直聘/猎聘数据（薪资行情数据支撑）
5. 微博职场热搜（实时热点）

## 各子方向写作侧重
| 子方向 | 核心情绪 | 主要结构 |
|--------|---------|---------|
| 裁员/失业 | 焦虑+共鸣+希望 | 故事→反思→重建 |
| 职场技巧 | 实用+顿悟感 | 痛点→方法→案例 |
| 副业变现 | 好奇+行动欲 | 案例→拆解→步骤 |
| 职场人际 | 共鸣+释然 | 场景→分析→建议 |
| AI+职场 | 危机+机会 | 警示→机会→行动 |

## 违规注意
- 不夸大收益（月入几万等需加免责说明）
- 不对具体公司/雇主进行负面点名
- 不做职业规划的绝对化承诺
