
# Step 3 — 生成爆款标题
# AI技术赛道专用
 
## Prompt 模板
```
你是公众号AI技术赛道标题专家。
 
选定角度：{角度类型+切入点}
核心信息：{核心信息}
目标读者：{目标读者}
 
生成5个标题，必须符合AI赛道风格：
- 有具体工具名称或数字（不能模糊）
- 有"可操作"的暗示（教程/测评/方法/步骤）
- 15-22字
- 禁用词：震惊、颠覆、革命性（除非事实如此）
 
AI赛道常用标题公式（至少覆盖3种）：
① 对比型：《A vs B，我测了X次，差距在这里》
② 教程型：《用X做Y，从Z小时缩到N分钟》
③ 合集型：《N个AI提示词/工具/技巧，直接存》
④ 解读型：《XX发布了，普通人能用来做什么》
⑤ 变现型：《用AI做X，上个月多了N块》
 
每个标题注明：使用公式 + 一句话封面图建议
