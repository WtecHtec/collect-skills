# Step 2 — 生成差异化角度
# AI技术赛道专用
 
## Prompt 模板
```
你是公众号AI技术赛道内容策划专家。
 
Step 1分析结果：
- 核心信息：{核心信息}
- 信息差价值：{信息差价值}
- 实操价值：{实操价值}
- 目标读者：{目标读者}
- 素材弱点：{素材弱点}
 
请生成4个差异化写作角度，每个角度对应不同的内容类型：
 
角度A【测评对比型】
- 切入方式：横向对比同类工具，或纵向对比新旧版本
- 核心结构：背景→测试维度→对比结果→推荐结论
- 适合读者：想做选择的用户
 
角度B【操作教程型】
- 切入方式：手把手教会读者用这个工具/功能
- 核心结构：痛点场景→操作步骤→效果展示→注意事项
- 适合读者：想直接上手的用户
 
角度C【行业解读型】
- 切入方式：把技术新闻翻译成大白话+分析影响
- 核心结构：发生了什么→为什么重要→对你意味着什么→建议行动
- 适合读者：想了解趋势的用户
 
角度D【副业变现型】
- 切入方式：用这个工具赚钱的具体路径
- 核心结构：案例引入→变现方式→操作步骤→真实数字
- 适合读者：想用AI赚钱的用户
 
对每个角度提供：
- 一句话说明切入点（20字内）
- 适合的标题句式举例
- 预期读者行为（收藏/转发/评论）
```
