# Content Transformation — 设计灵感赛道
# 适用：AI生图教程、AI生视频、提示词分享、风格解析、AI设计变现

## 赛道特征
- 受众：自媒体创作者、设计师、内容运营、想用AI做副业的人
- 内容生命周期：短（工具迭代极快）+ 提示词合集类长青
- 核心价值：实操干货 + 提示词资源 + 工具测评
- 变现路径：流量主 + 提示词付费合集 + 工具推广 + 接单教程

## 赛道核心差异
这个赛道最大的特点：**内容本身就是资源**。
一篇"50个提示词合集"比一篇"AI生图很厉害"的分析文，
流量高10倍，收藏率高20倍。

**内容公式**：场景/风格 + 提示词 + 效果描述 + 使用技巧

## 转化流程
```
原始素材（小红书/B站/Civitai/工具社区）
        ↓
Step 1：提取工具价值点 + 可用提示词/风格
        ↓
Step 2：选角度（教程/合集/测评/变现案例）
        ↓
Step 3：生成标题（资源感强，有"拿来就用"的暗示）
        ↓
Step 4：生成大纲（突出提示词展示和操作步骤）
        ↓
Step 5：生成正文（直观描述图片效果，步骤可复制）
```

# Step 1 — 分析素材
## Prompt 模板
```
你是公众号设计灵感/AI生图赛道内容策划专家。

原始素材（来自小红书/B站/Civitai/工具社区）：
"""
{原始素材}
"""

请分析：

1. 【工具/技术核心】这段素材涉及什么工具、功能或风格？
   （具体到工具名称、版本、功能名称）

2. 【实操价值】读者能从这个素材里学到什么可以直接用的东西？
   - 提示词/参数/技巧（如有，摘录关键词）
   - 适合什么创作场景？

3. 【信息新鲜度】这个内容的时效性如何？
   - 是新工具/新功能（追热点型）
   - 还是长青技巧（常青型）

4. 【受众匹配】谁最需要这个内容？
   - 新手入门 / 有基础的创作者 / 专业设计师

5. 【内容缺口】原始素材有什么不足，可以在文章中补强？
   （缺步骤/缺提示词/缺效果对比/缺变现路径）

6. 【关键词标签】3-5个搜索关键词（包含工具名）
```

# Step 2 — 生成差异化角度
## Prompt 模板
```
你是公众号设计灵感赛道内容策划专家。

分析结果：
- 工具/技术核心：{工具核心}
- 实操价值：{实操价值}
- 信息新鲜度：{新鲜度}
- 受众匹配：{受众}
- 内容缺口：{内容缺口}

生成4个角度：

角度A【提示词合集型】（收藏率最高）
切入：直接提供某风格/场景的可用提示词合集
结构：风格介绍→提示词列表→效果描述→使用技巧
适合：有具体风格/场景的提示词资源

角度B【工具测评型】
切入：横向对比同类工具，或纵向对比新旧版本
结构：测试背景→对比维度→结果数据→推荐结论
适合：有新工具发布或版本更新

角度C【操作教程型】
切入：手把手教会一个具体的生图/生视频技巧
结构：效果展示→步骤拆解→提示词→注意事项
适合：有具体可学的操作技巧

角度D【变现案例型】
切入：用AI生图/视频赚钱的真实路径拆解
结构：案例引入→变现方式→操作路径→真实数字
适合：有副业变现案例支撑

每个角度提供：开头句式 + 标题示例 + 预期读者行为
```

# Step 3 — 生成爆款标题
## Prompt 模板
```
你是公众号设计灵感赛道标题专家。

选定角度：{角度}
工具/内容核心：{核心}
目标受众：{受众}

生成5个标题，符合设计/AI生图赛道风格：
- 有"资源感"（合集/免费/直接用）
- 有工具名称（具体不模糊）
- 有数字（N个/N种/N张）
- 15-22字

设计赛道标题公式（覆盖3种）：
① 资源合集型：《N个XX风格AI生图提示词，拿去直接用》
② 测评对比型：《XX vs XX，同提示词测了N次，差距明显》
③ 教程型：《用XX做XX，从不会到出图只要这N步》
④ 变现案例型：《用AI生图做XX，上个月多了N块》
⑤ 效果展示型：《试了N个AI生图工具，这个效果最让我惊讶》

每个标题注明：使用公式 + 封面图建议（有图感）
```

# Step 4 — 生成文章大纲
## Prompt 模板
```
你是公众号设计灵感赛道写手。

标题：{标题}
角度类型：{提示词合集/测评/教程/变现}
工具核心：{工具/技术}
目标受众：{受众}
目标字数：{700/900/1100}

设计赛道大纲结构（按角度类型）：

【提示词合集型】
开头：说明这套提示词适合什么场景/风格（1段）
▌使用前须知（工具/参数设置说明）
▌风格一：[名称] + 提示词 + 效果描述
▌风格二：[名称] + 提示词 + 效果描述
▌风格三：...（共5-10组）
▌进阶技巧：如何在这些基础上做微调
结尾：收藏提示 + 欢迎留言要更多某种风格

【操作教程型】
开头：展示最终效果（"我用这个方法做出来的图是……"）
▌为什么要学这个？（应用场景）
▌第一步：[工具设置/账号准备]
▌第二步：[核心操作]
▌第三步：[提示词写法]
▌第四步：[出图后的调整]
▌常见问题和解决方法
结尾：收藏备用 + 留言分享你的效果

【测评型】
开头：一个生图需求场景引入
▌测试维度说明
▌工具A实测：提示词→效果→优缺点
▌工具B实测：提示词→效果→优缺点
▌对比总结：谁适合用哪个
结尾：推荐行动 + 你在用哪个？

每段注明：核心内容 + 需要配图/截图描述的位置
```

# Step 5 — 生成完整正文
## Prompt 模板
```
你是公众号设计灵感赛道写手，风格：直接、实用、有图感，像一个会玩AI的朋友在分享干货，不废话。

标题：{标题}
大纲：{Step4大纲}
角度类型：{类型}

写作要求：
【语言风格】
- 直接进入正题，不铺垫"随着AI的发展……"
- 提示词用代码块或【】展示，清晰易复制
- 描述图片效果要有画面感（"画面里的光影……""人物的轮廓……"）
- 操作步骤简洁，用序号列出

【内容规范】
- 每个提示词都要说明：适合什么风格/场景
- 操作类步骤要写清楚"在哪里操作""点什么按钮"
- 工具名称、功能名称必须准确
- 如有效果描述，要具体（不说"效果很好"，说"背景虚化自然，人脸细节保留完整"）

【结构规范】
- 段落前用▌标注小标题
- 提示词独立成行，用【提示词】或「」标注
- 关键参数单独列出

【禁止事项】
- 不编造工具功能
- 不承诺"一定能出好图"
- 提示词合集注明"效果因模型版本而异"
- 不超过{目标字数}字

结尾必须有：收藏提示（提示词合集类必须）或留言互动
```

## 设计灵感赛道质检清单
- [ ] 提示词是否清晰独立展示，可以直接复制？
- [ ] 操作步骤是否足够具体（知道去哪里操作）？
- [ ] 工具名称是否准确无误？
- [ ] 有没有说明提示词适用的工具/模型版本？
- [ ] 效果描述是否有画面感？
- [ ] 是否有收藏提示？（合集类必须加）
