# 01 - 选题策划阶段

本文档包含选题策划阶段（Step 1-5）的所有详细prompt模板。

---

## Step 1-2: 热点监测与筛选

### 使用场景
- 用户提供了具体热点/话题
- 用户只给了一个大致方向
- 用户完全没有想法

### Prompt模板

```
你是公众号选题策划专家。

我的公众号定位：
- 目标用户：26-35岁职场打工人
- 内容方向：职场成长、潜规则
- 调性：真实、扎心、有共鸣、有干货

热点/话题方向：
[用户提供的热点或话题]

任务：评估这个热点是否适合做选题，或者生成相关的5-10个选题方向。

评估维度：
1. 用户匹配度（1-10分）：是否是目标用户关心的
2. 传播潜力（1-10分）：话题讨论度、情绪共鸣
3. 切入角度：我们可以从什么独特角度切入
4. 风险评估（低/中/高）：是否涉及敏感话题、不实信息
5. 时效性（强/中/弱）：热度能持续多久

输出格式：
热点：[热点名称]
匹配度：X分  传播潜力：X分
切入角度：[具体角度]
风险：低/中/高
时效性：强/中/弱
推荐理由：[说明]

如果不适合，说明原因并建议替代方向。
如果适合，继续下一步生成具体选题。
```

### 输出示例

```
热点：职场35岁危机讨论
匹配度：9分  传播潜力：9分
切入角度：从打工人视角看35岁职场困境，不贩卖焦虑，而是给出实际应对方法
风险：低（避免性别对立，聚焦方法论）
时效性：中（长期话题，可以做）
推荐理由：完美匹配目标用户年龄段，痛点明确，有很强的情感共鸣空间

建议进入下一步生成具体选题方向。
```

## Step 3: 选题方向生成

### 使用场景
Step 2确认热点适合后，生成3个不同风格的选题供用户选择。

### Prompt模板

```
你是爆款选题策划专家。

热点：[Step 2确定的热点]
切入角度：[Step 2输出的角度]

任务：基于这个热点，生成3个不同风格的选题方向。

选题要求：
1. 标题：吸引但不标题党（20字内）
2. 核心痛点：击中26-35岁职场打工人的什么痛点
3. 内容角度：
   - 方向1：个人故事+经验分享（暖心共鸣型）
   - 方向2：深度分析+方法论（干货价值型）
   - 方向3：反鸡汤+扎心真相（犀利清醒型）
4. 预期传播：预估阅读量级别、转发潜力
5. 合规性：是否符合微信推荐规则

输出3个选题方向，每个包含：
- 标题（3个备选标题）
- 副标题/摘要
- 核心大纲（3-5个要点）
- 金句方向（3-5句）
- 配图建议
- 为什么这个角度能打动读者
```

### 输出示例

```
方向1：暖心共鸣型
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

标题备选：
1. 30岁那年，我终于不再害怕35岁
2. 从28岁到35岁，我用7年时间和解了焦虑
3. 致30岁的你：35岁不是终点，是新的开始

副标题：
一个30岁职场打工人，关于年龄焦虑的真实和解之路

核心大纲：
1. 28岁的恐慌：第一次听到"35岁危机"时的崩溃
2. 30岁的觉醒：意识到焦虑比年龄更可怕
3. 32岁的行动：开始建立自己的护城河
4. 35岁的平静：终于明白年龄只是数字
5. 写给焦虑中的你：3个具体建议

金句方向：
1. "真正毁掉你的，不是35岁，而是28岁就开始的焦虑。"
2. "年龄是递增的，但能力也可以是。"
3. "与其害怕35岁，不如从今天开始积累。"
4. "35岁危机的对面，是35岁的从容。"
5. "每个年龄都有它的美好，关键是你有没有准备好。"

配图建议：
- 封面：一个打工人背影，看着窗外，温暖的光线
- 文中：时间轴图、成长曲线图、温暖生活场景

为什么这个角度能打动读者：
用真实故事和情感共鸣，让读者感觉"这说的就是我"，同时给出希望和具体方法，不贩卖焦虑。

预期传播：5万+ 阅读，高转发率（易引发共鸣）
合规性：完全符合，正向价值观

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方向2：干货价值型
[类似结构，但聚焦实用方法论]

方向3：犀利清醒型
[类似结构，但风格更扎心、反鸡汤]
```

### 用户选择后的操作

询问用户："这3个方向你更喜欢哪一个？或者需要我调整哪个方向？"

等用户确认后，进入Step 4。

## Step 4: 提取关键词

### 使用场景
用户选定选题方向后，提取SEO关键词。

### Prompt模板

```
选题：[用户选定的选题]

任务：提取关键词用于SEO优化和内容创作。

请输出：
1. 核心关键词（3-5个）：如"职场"、"打工人成长"、"30岁"
2. 长尾关键词（5-10个）：如"30岁职场打工人焦虑"、"独立打工人成长"
3. 情绪关键词（3-5个）：如"焦虑"、"清醒"、"成长"
4. 相关话题标签：如#职场成长 #独立打工人 #30岁

这些关键词需要在文章中自然出现3-5次。
```

### 输出示例

```
选题：30岁那年，我终于不再害怕35岁

核心关键词（3-5个）：
- 30岁
- 35岁危机
- 职场打工人
- 年龄焦虑
- 个人成长

长尾关键词（5-10个）：
- 30岁职场打工人成长
- 35岁职场危机应对
- 打工人年龄焦虑
- 职场打工人如何面对年龄压力
- 30-35岁职业规划
- 职场打工人自我提升
- 克服年龄焦虑的方法
- 职场中年打工人成长

情绪关键词（3-5个）：
- 焦虑
- 恐慌
- 觉醒
- 平静
- 从容

相关话题标签：
#职场打工人 #30岁 #年龄焦虑 #个人成长 #职场成长

使用建议：
- 在文章标题、摘要、开头、结尾自然出现核心关键词
- 长尾关键词分散在文章各段落
- 情绪关键词用于描述心路历程
- 话题标签添加到文章发布设置中
```

## Step 5: 文章结构设计

### 使用场景
关键词提取完成后，设计完整的文章框架。

### Prompt模板

```
选题：[选题名称]
关键词：[关键词清单]
风格：[用户选定的风格：暖心共鸣型/干货价值型/犀利清醒型]

任务：设计文章结构框架

要求：
1. 文章长度：1000字以上（根据选题复杂度灵活调整，可以是1500字、3000字或更长）
2. 结构：
   - 开头（100-300字）：真实故事/场景引入，制造共鸣
   - 主体（根据内容需要）：3-5个核心观点，每个观点包含：
     * 小标题（吸引人的）
     * 故事/案例（200-300字）
     * 扎心真相/洞察（100-200字）
     * 实用方法/干货（200-300字）
   - 结尾（100-300字）：情感升华 + 互动引导

输出详细大纲：

标题：[最终标题 + 2个备选]

一、开头（500字）
场景/故事：[具体场景描述，要有画面感]
钩子：[吸引读者继续读下去的点]

二、主体部分

观点1：[小标题]
- 故事：[故事梗概，要真实可信]
- 真相：[核心洞察，要扎心]
- 方法：[3个可执行的具体建议]

观点2：[小标题]
[重复结构]

观点3：[小标题]
[重复结构]

三、结尾（500字）
- 情感升华方向：[如何升华主题]
- 互动话题：[什么问题能引发评论讨论]

四、文章金句（10句）
[列出10个可以用在文章中的金句]

五、配图建议（5-8张）
[每个部分需要什么样的配图]
```

### 输出示例

```
标题：30岁那年，我终于不再害怕35岁
备选1：从28岁到35岁，我用7年时间和解了焦虑
备选2：致30岁的你：35岁不是终点，是新的开始

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

一、开头（100-300字）

场景/故事：
28岁那年的某个深夜，我坐在出租屋的阳台上，看着对面写字楼依然灯火通明。手机里刚刷到一篇文章《35岁职场危机：你准备好了吗？》，我的手开始发抖。

距离35岁还有7年，但恐慌已经提前袭来。我开始疯狂焦虑：7年后我会不会被优化？会不会找不到工作？会不会一事无成？

那一夜，我失眠了。

钩子：
但7年后的今天，35岁的我，终于明白：真正毁掉你的，不是35岁，而是28岁就开始的焦虑。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

二、主体部分

观点1：28岁的恐慌——被"35岁危机"绑架的青春

故事：
28岁的我，每天都活在对未来的恐慌中。看到同龄人晋升，我焦虑；听到裁员消息，我焦虑；刷到35岁职场危机的文章，我更焦虑。

我开始疯狂考证：考了CPA、PMP、ACCA...恨不得把所有证书都考一遍。我开始疯狂学习：报了10个线上课程，但没有一个学完的。

我以为这样就能对抗35岁危机，但其实，我只是在用"伪努力"掩盖真正的恐慌。

真相：
焦虑的本质，不是对35岁的恐惧，而是对当下的不满。

你害怕35岁，是因为你现在28岁，却没有28岁该有的成长。你害怕未来没有竞争力，是因为你现在就缺乏核心竞争力。

年龄焦虑，是用"未来可能的危机"，逃避"当下必须的改变"。

方法：
1. 停止"伪努力"：考10个证书不如深耕1个领域。问自己：这些努力，是在建立护城河，还是在缓解焦虑？

2. 建立"核心竞争力"清单：列出你的3个核心能力，每天花1小时深度打磨，而不是广撒网浅尝辄止。

3. 用"成长日记"对抗焦虑：每天记录1个进步，哪怕很小。3个月后回看，你会发现成长比焦虑更真实。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

观点2：30岁的觉醒——意识到焦虑比年龄更可怕

[类似结构]

观点3：32岁的行动——开始建立自己的护城河

[类似结构]

观点4：35岁的平静——终于明白年龄只是数字

[类似结构]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

三、结尾（100-300字）

情感升华方向：
从害怕35岁，到拥抱35岁，我用了7年。

这7年，我学会的最重要的事是：年龄是递增的，但能力也可以是。

35岁不是终点，而是人生的新起点。如果你30岁，请不要浪费时间焦虑35岁；如果你25岁，请不要提前透支30岁的恐慌。

每个年龄都有它的美好，关键是你有没有准备好。

互动话题：
你在什么年龄第一次感受到年龄焦虑？又是如何走出来的？欢迎在评论区分享你的故事，我们一起成长。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

四、文章金句（10句）

1. 真正毁掉你的，不是35岁，而是28岁就开始的焦虑。
2. 年龄焦虑的本质，是用未来可能的危机，逃避当下必须的改变。
3. 考10个证书，不如深耕1个领域。
4. 年龄是递增的，但能力也可以是。
5. 35岁危机的对面，是35岁的从容。
6. 每个年龄都有它的美好，关键是你有没有准备好。
7. 停止伪努力，开始真成长。
8. 护城河不是一天挖成的，但每天挖一点，7年后就是一条护城河。
9. 与其害怕35岁，不如从今天开始积累。
10. 30岁的你，决定了35岁的你。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

五、配图建议（5-8张）

1. 封面图：打工人背影，看着窗外，温暖的光线，象征思考和成长
2. 开头配图：深夜的写字楼，渲染28岁的焦虑氛围
3. 观点1配图：杂乱的书桌，一堆证书和课程，象征"伪努力"
4. 观点2配图：打工人在咖啡厅，笔记本上写着目标，象征"觉醒"
5. 观点3配图：打工人专注工作的侧影，象征"行动"
6. 观点4配图：打工人自信的笑容，象征"平静和从容"
7. 结尾配图：打工人站在高处看风景，象征"新起点"
```

### 输出后的操作

询问用户："这个结构你满意吗？需要调整哪里？"

确认后告诉用户："结构确认！接下来我会用AI生成完整初稿，大约需要15-20分钟。生成后，最关键的Step 7需要你花60-90分钟进行人工优化。"

然后进入Step 6。
