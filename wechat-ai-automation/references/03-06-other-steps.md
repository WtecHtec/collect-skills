# 04 - 发布优化阶段

## Step 10: 标题优化

### Prompt模板

```
文章主题：[主题]
目标用户：26-35岁职场女性

任务：生成10个不同风格的标题

标题类型（每种2个）：
1. 数字型：包含数字，如"3年时间，我从月薪5K到年薪30W"
2. 疑问型：引发好奇，如"30岁还没成功，是我不够努力吗？"
3. 对比型：强烈对比，如"同龄人年薪50W，我月薪5K，差距在哪？"
4. 共鸣型：情感共鸣，如"30岁的你，是不是也在假装坚强"
5. 冲突型：观点冲突，如"那些职场鸡汤，差点毁了我的30岁"

每个标题：
- 长度：15-25字
- 吸引但不标题党
- 包含关键词：[关键词]
- 符合账号调性

每个标题后面给出：
- 预估点击率（1-10分）
- 推荐理由（为什么这个标题能吸引点击）

最后推荐得分最高的前3个。
```

## Step 11: 发布前检查清单

```
□ 内容检查
  □ 全文通读，检查错别字
  □ 逻辑连贯
  □ 金句突出
  □ 关键词自然融入

□ 合规检查
  □ 无不实信息
  □ 无敏感话题
  □ 无低俗擦边
  □ 符合推荐规则

□ 视觉检查
  □ 配图清晰无水印
  □ 排版美观
  □ 手机端预览

□ 技术设置
  □ 原创声明
  □ 话题标签
  □ 文章摘要
  □ 封面图
  □ 推送时间

□ 互动设置
  □ 互动话题
  □ 评论区开启
  □ 赞赏开启
```

## Step 12: 发布后监测

前2小时监测重点：
- 每10分钟看阅读量
- 观察分享数、收藏数
- 及时回复评论
- 处理异常情况

# 06 - 补充功能

## 评论管理

```
这是文章的评论：
[评论列表]

请帮我：
1. 分类：好评/中评/差评/提问
2. 为每条生成回复话术（30-50字，真诚温暖）
3. 找出3-5条优质评论建议置顶
4. 提取用户关心的问题作为选题方向

回复要求：真诚、温暖、专业
```

## SEO优化

```
文章主题：[主题]
核心关键词：[关键词]

请优化SEO：
1. 写80字文章摘要（含关键词，吸引点击）
2. 建议5个话题标签
3. 关键词布局建议（不影响阅读）
```

## 多平台分发

```
这是公众号文章（4000字）。

请改编成：
1. 小红书版本（1000字 + 10个金句）
2. 知乎版本（更客观理性）
3. 微博版本（300字精华 + 引流钩子）

保持核心观点一致，适配平台特点。
```
