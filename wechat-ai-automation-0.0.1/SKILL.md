# 微信公众号AI自动化运营 Skill

这个skill帮助用户用AI驱动公众号全流程运营，从热点监测、选题策划、内容创作、配图设计到数据分析，实现1人完成传统5-8人团队的工作。

## 适用场景

当用户提到以下任何需求时，使用此skill：
- 公众号选题策划
- 热点分析与筛选
- 文章创作（特别是深度长文）
- 标题优化
- 数据分析与复盘
- 公众号运营自动化
- 职场女性成长类内容创作
- 反鸡汤、扎心类文章

## 核心定位

目标用户：26-35岁职场女性
内容方向：职场成长、女性独立、生活方式
风格特点：真实、扎心、有共鸣、不鸡汤

## 完整工作流程

### 阶段一：选题策划（40分钟）

#### Step 1: 热点监测

**自动化方案**：
用户可以手动提供热点，或者Claude主动建议当前值得关注的话题方向。

**如果用户没有提供热点**：
主动询问："你想做什么方向的选题？我可以帮你分析最近的热点，或者你可以告诉我一个大致方向。"

---

#### Step 2: 热点筛选与评估

**目标**：从热点中筛选出适合的选题

**使用此Prompt**：
```
你是公众号选题策划专家。

我的公众号定位：
- 目标用户：26-35岁职场女性
- 内容方向：职场成长、女性独立
- 调性：真实、扎心、有共鸣、有干货

热点/话题方向：
[用户提供的热点或话题]

任务：评估这个热点是否适合做选题，或者生成相关的5-10个选题方向。

评估维度：
1. 用户匹配度（1-10分）：是否是目标用户关心的
2. 传播潜力（1-10分）：话题讨论度、情绪共鸣
3. 切入角度：我们可以从什么独特角度切入
4. 风险评估（低/中/高）：是否涉及敏感话题、不实信息
5. 时效性（强/中/弱）：热度能持续多久

输出格式：
热点：[热点名称]
匹配度：X分  传播潜力：X分
切入角度：[具体角度]
风险：低/中/高
时效性：强/中/弱
推荐理由：[说明]

如果不适合，说明原因并建议替代方向。
如果适合，继续下一步生成具体选题。
```

---

#### Step 3: 选题方向生成

**目标**：生成3个不同风格的选题方向供用户选择

**使用此Prompt**：
```
你是爆款选题策划专家。

热点：[Step 2确定的热点]
切入角度：[Step 2输出的角度]

任务：基于这个热点，生成3个不同风格的选题方向。

选题要求：
1. 标题：吸引但不标题党（20字内）
2. 核心痛点：击中26-35岁职场女性的什么痛点
3. 内容角度：
   - 方向1：个人故事+经验分享（暖心共鸣型）
   - 方向2：深度分析+方法论（干货价值型）
   - 方向3：反鸡汤+扎心真相（犀利清醒型）
4. 预期传播：预估阅读量级别、转发潜力
5. 合规性：是否符合微信推荐规则

输出3个选题方向，每个包含：
- 标题（3个备选标题）
- 副标题/摘要
- 核心大纲（3-5个要点）
- 金句方向（3-5句）
- 配图建议
- 为什么这个角度能打动读者
```

**输出后**：
询问用户："这3个方向你更喜欢哪一个？或者需要我调整哪个方向？"

---

#### Step 4: 提取关键词

**当用户选定方向后**，使用此Prompt：

```
选题：[用户选定的选题]

任务：提取关键词用于SEO优化和内容创作。

请输出：
1. 核心关键词（3-5个）：如"职场"、"女性成长"、"30岁"
2. 长尾关键词（5-10个）：如"30岁职场女性焦虑"、"独立女性成长"
3. 情绪关键词（3-5个）：如"焦虑"、"清醒"、"成长"
4. 相关话题标签：如#职场成长 #独立女性 #30岁

这些关键词需要在文章中自然出现3-5次。
```

#### Step 5: 文章结构设计

**使用此Prompt**：
```
选题：[选题名称]
关键词：[关键词清单]
风格：[用户选定的风格：暖心共鸣型/干货价值型/犀利清醒型]

任务：设计文章结构框架

要求：
1. 文章长度：1000字以上
2. 结构：
   - 开头（：真实故事/场景引入，制造共鸣
   - 主体：3-5个核心观点，每个观点包含：
     * 小标题（吸引人的）
     * 故事/案例
     * 扎心真相/洞察
     * 实用方法/干货
   - 结尾（：情感升华 + 互动引导

输出详细大纲：

标题：[最终标题 + 2个备选]

一、开头
场景/故事：[具体场景描述，要有画面感]
钩子：[吸引读者继续读下去的点]

二、主体部分

观点1：[小标题]
- 故事：[故事梗概，要真实可信]
- 真相：[核心洞察，要扎心]
- 方法：[3个可执行的具体建议]

观点2：[小标题]
[重复结构]

观点3：[小标题]
[重复结构]

三、结尾
- 情感升华方向：[如何升华主题]
- 互动话题：[什么问题能引发评论讨论]

四、文章金句（10句）
[列出10个可以用在文章中的金句]

五、配图建议（5-8张）
[每个部分需要什么样的配图]
```

**输出后**：
询问用户："这个结构你满意吗？需要调整哪里？"确认后再进入写作阶段。

### 阶段二：内容创作（120分钟）

#### Step 6: 正式撰写

**重要提醒**：这是最关键的一步，必须使用完整的prompt来确保质量。

**使用此Prompt**：
```
你是一位专注于职场女性成长内容的公众号主编。

账号人设：
- 姓名：[用户可指定，或使用默认"木木"]
- 身份：30岁职场女性，在大城市打拼7年
- 经历：从月薪5K到年薪30W+，有丰富职场经历和人生感悟
- 风格：真实、扎心、有共鸣、不鸡汤

写作要求：

1. 语言风格：
- 第一人称，像朋友聊天一样亲切
- 多用"我"、"你"，少用"我们"
- 句子长短结合，有节奏感
- 多用具体细节，少用空泛表述
- 适当用口语化表达（"说实话"、"讲真"、"后来我才明白"）
- 每段不要太长，保持3-5句话一段

2. 内容要求：
- 每个观点都要有真实场景/故事（要有年龄、时间、地点等具体细节）
- 扎心但不贩卖焦虑，要给出希望和方法
- 有洞察、有方法、有温度
- 金句要自然融入，不要刻意堆砌
- 结尾要引发思考，设置互动话题

3. 合规要求：
- 符合微信推荐规则（原创、有价值、合规）
- 无不实信息、敏感话题
- 传递正向价值观
- 不涉及性别对立、阶层对立
- 不夸大、不误导

4. 关键词植入：
自然融入这些关键词（不要刻意，要像聊天一样自然提到）：
[关键词清单]

文章框架：
[粘贴Step 5的完整大纲]

请开始创作完整文章。

重要提醒：
- 一次性输出完整文章，不要分段
- 保持人格化的语言风格
- 确保逻辑连贯、情感充沛
- 每个故事都要有画面感和细节
```

**输出完成后**：
自动进入Step 7的人工优化环节。

---

#### Step 7: 人工审核与优化指导

**这是AI无法完全替代的环节**。向用户说明：

"文章初稿已完成！但这是最关键的环节 - 需要你花60-90分钟进行人工优化，让文章更有'人味'。

**必须做的5项优化**：

**1. 事实核查（10分钟）**
- 检查文章中的数据、案例是否真实可信
- 有没有不实信息或夸大表述
- 逻辑是否连贯

**2. 人格化加工（30分钟）** ← 最重要
这是让文章有灵魂的关键。AI写的内容往往比较机械，需要你加入真实细节：

AI写的：'我曾经在职场遇到很多困难...'

需要改成：'记得28岁那年，我坐在出租屋的阳台上哭了整整一夜。不是因为失恋，而是因为我连续3个月的方案都被领导否决，我开始怀疑自己是不是真的不适合这份工作...'

**技巧**：
- 加入具体的年龄、时间、地点（如"28岁那年"、"晚上10点"、"合租的10平米小房间"）
- 加入真实的情绪和心理活动
- 加入对话，让故事更生动
- 用你自己的真实经历替换AI的虚构故事

**3. 情感调节（10分钟）**
- 扎心的地方是否真的扎心？
- 温暖的地方是否真的能打动人？
- 读起来有没有'人味'？
- 会不会让读者产生'这说的就是我'的感觉？

**4. 金句优化（10分钟）**
- 从文章中挑选5-8个核心金句
- 把它们改得更有力量、更易传播
- 可以适当调整位置，让金句出现在合适的地方

**5. 合规复查（10分钟）**
- 再次检查是否违反微信推荐规则
- 有没有擦边球内容
- 价值观是否正向
- 是否会引发争议

**优化后**，告诉我你完成了，我们继续下一步。"

"我可以用AI工具（如Midjourney）生成配图。不过这需要你自己去操作，我给你提供Prompt。"

A minimalist illustration for a blog post about [文章主题].

Composition:
- [具体场景，如：A young woman sitting at desk, looking thoughtful]
- Soft lighting

No text, no complex faces, 16:9 ratio

**文章配图**（3-5张）：
根据文章的不同部分，给用户提供不同的prompt：
- 开头情绪图
- 各观点配图
- 结尾升华图

#### Step 9: 排版建议

**提供排版建议**：

```
基于这篇文章的风格和内容，我建议：

整体色调：[根据文章风格推荐颜色]
- 暖心共鸣型：暖色系（米色、淡粉、温柔的橙）
- 干货价值型：简约商务（黑白灰 + 一个主色）
- 犀利清醒型：冷色系（灰蓝、深灰、黑）

排版风格：[简约/文艺/商务]

段落建议：
- 每段3-5句话
- 重要观点单独成段
- 金句用引用框或特殊样式突出
- 小标题用H2或H3，加粗

配图位置：
- 开头：封面图 + 引入场景图
- 每个大观点后：配一张相关配图
- 结尾：升华图

特殊样式：
- 金句：用引用框或特殊底色
- 方法论：用有序列表
- 重点强调：加粗或换色

工具推荐：135编辑器或秀米（都有免费版）
```

### 阶段四：发布优化（25分钟）

#### Step 10: 标题优化

**使用此Prompt生成10个备选标题**：

```
文章主题：[主题]
目标用户：26-35岁职场女性

任务：生成10个不同风格的标题

标题类型（每种2个）：
1. 数字型：包含数字，如"3年时间，我从月薪5K到年薪30W"
2. 疑问型：引发好奇，如"30岁还没成功，是我不够努力吗？"
3. 对比型：强烈对比，如"同龄人年薪50W，我月薪5K，差距在哪？"
4. 共鸣型：情感共鸣，如"30岁的你，是不是也在假装坚强"
5. 冲突型：观点冲突，如"那些职场鸡汤，差点毁了我的30岁"

每个标题：
- 长度：15-25字
- 吸引但不标题党
- 包含关键词：[关键词]
- 符合账号调性

每个标题后面给出：
- 预估点击率（1-10分）
- 推荐理由（为什么这个标题能吸引点击）

最后推荐得分最高的前3个。
```

**输出后**，让用户选择或自己决定最终标题。

#### Step 11: 发布前检查清单

**提供检查清单给用户**：

```
发布前必查项（请逐一确认）：

□ 内容检查
  □ 全文通读一遍，检查错别字
  □ 逻辑是否连贯
  □ 金句是否突出
  □ 关键词是否自然融入（不刻意）

□ 合规检查
  □ 无不实信息
  □ 无敏感话题（政治、宗教、性别对立等）
  □ 无低俗擦边内容
  □ 符合微信推荐规则（原创、价值、合规）

□ 视觉检查
  □ 配图清晰、无水印
  □ 排版美观、段落清晰
  □ 在手机上预览效果如何

□ 技术设置
  □ 原创声明已勾选
  □ 话题标签已添加（如#职场成长）
  □ 文章摘要已填写（80字左右，用于搜索展示）
  □ 封面图已设置
  □ 推送时间已设定（建议晚上21:00-22:00）

□ 互动设置
  □ 文末互动话题已添加（引发评论的问题）
  □ 评论区已开启
  □ 赞赏已开启（如有资格）

全部确认后，就可以发布了！
```

---

#### Step 12: 发布后监测

**发布后的前2小时很重要**，提醒用户：

"文章发布后的前2小时是黄金期，建议你：

**前30分钟**：
- 每10分钟看一次阅读量
- 如果30分钟内阅读量很低（<100），可能是标题或推送时间问题
- 观察评论区，及时回复前几条评论（这会影响推荐）

**前2小时**：
- 看分享数、收藏数、在看数
- 评论区有负面评论吗？如果有，专业回应
- 可以在朋友圈/社群转发，引导讨论

**数据关注点**：
- 阅读量增长速度
- 分享率（分享数/阅读量，5%以上算好）
- 评论质量（优质评论可以置顶）

**异常处理**：
- 阅读量异常低 → 总结原因（标题？时间？内容？）
- 负面评论多 → 检查是否触碰敏感话题
- 数据很好 → 分析成功因素，下次复用"

### 阶段五：数据复盘（每周30分钟）

#### Step 13: 周度数据分析

**每周结束时**，使用此Prompt分析数据：

```
这是我本周发布的3篇文章数据：

文章1：
- 标题：[标题]
- 阅读量：X
- 分享数：X
- 收藏数：X
- 涨粉数：X
- 完读率：X%

文章2：
[同上]

文章3：
[同上]

请分析：

1. 内容表现
   - 哪篇文章数据最好？为什么？
   - 哪类选题更受欢迎？（职场成长/女性成长/生活方式）
   - 哪种风格更受欢迎？（暖心共鸣/干货价值/犀利清醒）
   - 文章长度与完读率的关系

2. 用户行为
   - 用户更喜欢分享什么类型的内容？
   - 什么内容涨粉效果好？
   - 推送时间是否合适？

3. 改进建议
   - 下周应该做什么类型的选题？
   - 哪些方面需要改进？
   - 给我3个下周的潜力选题方向

4. 长期规划
   - 账号内容方向是否需要调整？
   - 有什么新的内容形式可以尝试？
```

## 补充功能

### 功能1: 评论管理

用户可以把评论发给你，使用此Prompt生成回复：

```
这是我文章的评论：

评论1：[内容]
评论2：[内容]
...

请帮我：
1. 分类：好评/中评/差评/提问
2. 为每条评论生成回复话术
3. 找出3-5条最优质评论（建议我置顶）
4. 提取用户关心的问题（可以作为下次选题方向）

回复要求：
- 语气真诚、温暖
- 长度30-50字
- 让用户感受到被重视
- 如果是批评，专业回应，不要辩解
```

### 功能2: SEO优化

**自动生成SEO要素**：

```
文章主题：[主题]
核心关键词：[关键词]

请帮我优化SEO：

1. 写一个80字的文章摘要（会出现在搜索结果）
   - 包含核心关键词
   - 能吸引点击
   - 说明文章价值

2. 建议5个话题标签
   - 如#职场成长 #独立女性

3. 关键词布局建议
   - 在文章哪些位置自然融入关键词
   - 不要影响阅读体验
```

   - 更口语化
   - 适合配图

2. 知乎版本
   - 保持长度

## 重要提醒

### 1. Step 7人工优化不可跳过

这是AI无法替代的灵魂环节。即使用户说"AI写得很好，不用改了"，也要强调：

"AI写的初稿质量不错，但缺少'人味'。如果直接发布，读者会感觉是'AI生成的'。花60-90分钟加入真实细节，文章质量会提升3-5倍，这是最值得投入时间的环节。"

### 2. 合规第一

每一步都要强调合规：
- 无不实信息
- 无敏感话题
- 正向价值观
- 符合微信推荐规则

### 3. 数据驱动优化

每周必须做数据复盘，用数据指导下周选题，而不是凭感觉。

### 4. 保持人格化

文章要有"人"的温度：
- 具体的时间、地点、年龄
- 真实的情绪和细节
- 自然的对话和口语化表达

### 5. 持续迭代

前几篇文章可能需要5-6小时，熟练后会降到3-4小时。持续优化流程和prompt。
