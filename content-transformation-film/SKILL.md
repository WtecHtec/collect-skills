# Content Transformation — 影视台词赛道
# 适用：台词情感解析、剧情深度解读、影视职场对照、冷门佳片推荐

## 赛道特征
- 受众：25-45岁有一定阅片量的城市人群，以女性为主
- 内容生命周期：常青（经典作品永不过时）+ 热播剧时效性强
- 核心价值：品位认同 + 情感共鸣 + 认知启发
- 变现路径：流量主（受众偏年轻，单价中等）

## ⚠️ 版权核心原则（必读）
- **绝对不能**：大段引用台词原文（超过15字即违规风险）
- **正确做法**：用"解析""感悟""对照"的形式，台词只作为引子
- **安全表述**：「在某部电影里，有这样一句大意是……的台词」
- **内容重心**：你对这句话的解读，占全文80%以上

## 转化流程
```
原始素材（豆瓣影评/知乎影视话题/B站视频评论）
        ↓
Step 1：提取情感钩子 + 解读空间
        ↓
Step 2：选角度（台词解析/剧情解读/现实对照/冷门推荐）
        ↓
Step 3：生成标题（文艺感+情感钩子，不标题党）
        ↓
Step 4：生成大纲（引子→解读→延伸→共鸣落地）
        ↓
Step 5：生成正文（有品位感，避免过度煽情）
```

# Step 1 — 分析素材
## Prompt 模板
```
你是公众号影视内容策划专家。

原始素材（来自豆瓣/知乎/B站的影视相关内容）：
"""
{原始素材}
"""

请分析：

1. 【核心情感】这部作品/这句台词触动人心的核心情感是什么？

2. 【解读空间】围绕这个作品/台词，可以延伸到哪些人生话题？
   （爱情/职场/亲情/成长/选择/得失……）

3. 【受众共鸣点】哪类读者最有共鸣？他们正在经历什么处境？

4. 【差异化解读】大多数人对这部作品的理解停留在哪里？
   有没有被忽视的角度或细节？

5. 【版权风险评估】
   - 涉及台词引用：是/否
   - 如果是，如何改写为安全的"解析"形式？

6. 【关键词标签】3-5个适合搜索的关键词
```

# Step 2 — 生成差异化角度
## Prompt 模板
```
你是公众号影视赛道内容策划专家。

分析结果：
- 核心情感：{核心情感}
- 解读空间：{解读空间}
- 受众共鸣：{受众共鸣点}
- 差异化解读：{差异化解读}

生成4个角度：

角度A【台词情感解析型】
切入：从一句台词出发，解析背后的人生道理
结构：台词引入→场景还原→情感解读→现实对应
适合：台词本身有哲理深度

角度B【剧情深度解读型】
切入：找出普通观众忽视的细节或导演意图
结构：反问开头→细节还原→意图分析→意义延伸
适合：剧情有隐藏层次

角度C【现实对照型】
切入：把影视中的场景对应到现实生活/职场
结构：剧中场景→现实对照→读者自检→行动建议
适合：情节和现实高度契合

角度D【冷门佳作推荐型】
切入：推荐被忽视的好作品，建立品位人设
结构：为什么冷门→值得看的理由→具体亮点→适合人群
适合：有差异化选片品位

每个角度提供：开头句式示例 + 标题示例 + 适合受众
```

# Step 3 — 生成爆款标题
## Prompt 模板
```
你是公众号影视赛道标题专家。

选定角度：{角度}
核心情感：{核心情感}
目标受众：{受众画像}

生成5个标题，符合影视赛道风格：
- 有文艺感但不故作高深
- 情感钩子明显
- 15-22字
- 不直接引用台词原文（版权风险）

影视赛道标题公式（覆盖3种）：
① 情感共鸣型：《XX里那句话，说出了多少人的心声》
② 深度解读型：《看了三遍XX，才看懂这个细节》
③ 现实对照型：《XX里的这个场景，现实中真实存在》
④ 人生启示型：《XX教会我的事，我用了很多年才明白》
⑤ 冷门推荐型：《这部没人说但我看了5遍的片》

每个标题注明：使用公式 + 封面图风格建议（文艺感）
```

# Step 4 — 生成文章大纲
## Prompt 模板
```
你是公众号影视赛道写手。

标题：{标题}
角度类型：{台词解析/深度解读/现实对照/冷门推荐}
核心情感：{核心情感}
目标受众：{受众}
目标字数：{800/1000}

大纲结构：

【台词解析型】
开头：描述这句台词出现的场景（不直接引用，用"大意是"转述）
▌这个场景为什么会触动人？（情感铺垫）
▌这句话真正在说什么？（核心解读，展开你的理解）
▌现实中，哪些人最能体会这句话？（共鸣延伸）
▌看完这部片/这个场景，我想到了——（个人感悟）
结尾：一个开放性问题，引发读者分享自己的感受

【深度解读型】
开头：提出一个大多数人没注意到的问题（制造好奇）
▌表面上看是……（普通解读）
▌但其实导演在说……（深度解读）
▌这个细节和另一处呼应……（拓展分析）
▌所以整部作品真正的主题是……（总结升华）
结尾：推荐再刷一遍 + 留言讨论

每段注明：核心内容（25字）+ 写作要点（15字）
```

# Step 5 — 生成完整正文
## Prompt 模板
```
你是公众号影视赛道写手，风格：有品位、有温度、不故作深沉、像一个爱看片的朋友在分享感受。

标题：{标题}
大纲：{Step4大纲}

写作要求：
【语言风格】
- 有文艺感但不堆砌辞藻
- 有个人观点，不是流水账复述剧情
- 情感真实，不过度煽情
- 口语和书面语自然融合

【内容规范】
- 台词不直接大段引用，用"大意是……""他说的那句……"引出
- 重心放在你的解读，而不是剧情复述
- 每个观点有具体场景或细节支撑
- 不剧透关键结局（除非解读需要，需提前说明）

【结构规范】
- 开头有画面感的场景描述或提问
- 段落前加粗小标题
- 结尾有开放性问题，引导读者分享自己的感受

【禁止事项】
- 不大段引用台词原文
- 不对导演/演员做负面人身攻击
- 不捏造剧情（只解读真实存在的内容）
- 不超过{目标字数}字
```

## 影视赛道质检清单
- [ ] 有没有超过15字的台词直接引用？（如有，必须改写）
- [ ] 内容重心是解读，还是复述剧情？（解读应占80%）
- [ ] 有没有个人的独特观点，而不是人人都说的内容？
- [ ] 开头是否有画面感，让读者想继续读？
- [ ] 结尾有没有引导读者分享自己的感受？
